---
title: "Errors"
description: "Errors represent failed (non-200) responses from endpoints."
---

An error has:

- An HTTP status code
- A body type _(Optional)_

```yaml user.yml
errors:
  UserNotFoundError:
    status-code: 404
    type: UserNotFoundErrorBody

types:
  UserNotFoundErrorBody:
    properties:
      requestedUserId: string
```
