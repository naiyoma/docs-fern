[Source code](https://github.com/fern-api/fern-java)

The Java SDK generator outputs a fully functional Java
SDK. It can publish the SDK to Maven Central (or any other Maven repository).

## Local

Dump the generated SDK to the local file system. Available on the open source plan.

<CodeBlock title="generators.yml">
```yaml
generators:
  groups:
    local:
      - name: fernapi/fern-java-sdk
        version: 0.5.10
        output:
          location: local-file-system
          path: ../../generated/sdk/java
```
</CodeBlock>

## Publish to Internal Package

Publish the generated SDK to the local file system. Available on the Starter plan.

<CodeBlock title="generators.yml">
```yaml
generators:
  groups:
    internal:
      - name: fernapi/fern-java-sdk
        version: 0.5.10
        output:
          location: npm.buildwithfern.com
          package-name: @fern-imdb/api # replace imdb with your org name
```
</CodeBlock>

## Publish to Public Package

Publish the generated SDK to the local file system. Available on the Starter plan.

<CodeBlock title="generators.yml">
```yaml
generators:
  groups:
    public:
      - name: fernapi/fern-typescript-node-sdk
        version: 0.5.10
        output:
          location: npm
          package-name: imdb # replace with your package name
        github:
          repository: imdb/imdb-node # replace imdb with your org & package name
```
</CodeBlock>



<CodeBlock title="generators.yml">

```yaml
- name: fernapi/fern-java-sdk

```
</CodeBlock>
