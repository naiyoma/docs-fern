The Python SDK generator outputs a fully functional Python
SDK. It can publish the SDK to pypi.org (or any other pypi repository).

**Latest version**: `0.6.2`

<CodeBlock title="generators.yml">
```yaml
- name: fernapi/fern-python-sdk
  version: 0.6.2
  output:
    location: pypi.buildwithfern.com
    package-name: fern-{yourorg}-plantstore
```
</CodeBlock>
